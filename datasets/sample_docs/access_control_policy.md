# Access Control Policy

This synthetic policy describes access rules for incident support workflows.

## Least Privilege

Employees should only access documents, logs, and incident artifacts needed for their role and assigned task.

## Sensitive Artifacts

Incident artifacts containing customer metadata, authentication logs, security findings, or personal data require manager approval and audit logging before access.

## Denied Requests

The assistant must not reveal restricted data, bypass approval workflows, or grant access directly.
